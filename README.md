# Job Market Analysis

---

## Overview
This project focuses on analyzing job market data to extract meaningful insights related to job roles, skill demand, and salary trends.  
The analysis demonstrates proper handling of raw data, data cleaning practices, and analytical thinking to support career and business decision-making.

---

## Dataset

- **Raw Data:** Stored in `data/raw/`
- The raw dataset is kept unchanged to reflect real-world data conditions.
- Data contains job titles, skills, experience levels, locations, and salary-related information.

- **Cleaned Data:** Stored in `data/cleaned/`
- Cleaned dataset is prepared after handling missing values, duplicates, and inconsistencies.
- Used for analysis, SQL queries, and visualization.

---

## Tools & Technologies

- **Excel** – Initial data review and basic cleaning  
- **SQL** – Data querying, filtering, and analytical logic  
- **Power BI** – Data visualization and dashboard creation  
- **Microsoft Word** – Final project documentation and reporting  

---

## Project Workflow

1. Collected and reviewed raw job market data  
2. Identified missing values and inconsistent entries  
3. Performed data cleaning and transformations  
4. Analyzed job trends, skills demand, and salary patterns  
5. Prepared insights for reporting and visualization  

---

## Key Analysis Areas

- Job role distribution analysis  
- In-demand skills identification  
- Salary trend comparison across roles  
- Industry-wise job opportunities  

---

## Key Insights

- Certain technical and analytical skills show consistently high demand  
- Salary levels vary significantly based on role, experience, and skill set  
- Data-driven insights can help job seekers align skills with market demand  

*(Detailed insights are explained in the project report)*

## Challenges Faced

- Handling missing and inconsistent job data  
- Standardizing job titles and skill names  
- Cleaning real-world datasets with duplicates
